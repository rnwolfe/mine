Print version, commit hash, and build date.

## Usage

```bash
mine version
```

## Example Output

```
mine version 0.1.0
commit: a1b2c3d
built: 2026-02-14T10:30:00Z
```

## Details

- **version**: Semantic version (e.g., `0.1.0`)
- **commit**: Git commit hash at build time
- **built**: ISO 8601 timestamp of when the binary was built
